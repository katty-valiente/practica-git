GIT CLONE: permite copiar repositorios existentes y trabajar únicamente con la versión clonada GIT PULL: se emplea para extraer y descargar contenido desde un repositorio remoto y actualizar al instante el repositorio local para reflejar ese contenido GIT CHECKOUT: comando de Git que permite navegar entre diferentes ramas GIT LOG: es una herramienta básica de Git para explorar el historial del repositorio GIT BRANCH: te permite crear, enumerar y eliminar ramas, así como cambiar su nombre GIT ADD: el comando añade recursivamente los archivos que están dentro de él. GIT COMMIT: captura una instantánea de los cambios preparados en ese momento del proyecto. GIT PUSH: se utiliza para comunicar con otro repositorio, calcular lo que tu base de datos local tiene que la remota no tiene GIT MERGE: permite tomar las líneas independientes de desarrollo creadas e integrarlas en una sola rama
